# Environment Variables

Este documento centraliza as variaveis de ambiente do projeto.

## Basico (obrigatorio)

- `EVOLUTION_BASE_URL`
- `EVOLUTION_API_KEY`
- `EVOLUTION_INSTANCE`
- `CAMPAIGN_SEND_LOCK`
- `CAMPAIGN_DRY_RUN`
- `CONTACTS_CSV_PATH` (modo CSV local)

## Perfis operacionais

### Teste controlado

- `CAMPAIGN_SEND_LOCK=false`
- `CAMPAIGN_DRY_RUN=false`
- `CAMPAIGN_FORCE_PHONE=5562...`
- `CAMPAIGN_MAX_CONTACTS=1..20`

### Producao

- `CAMPAIGN_SEND_LOCK=false`
- `CAMPAIGN_DRY_RUN=false`
- `CAMPAIGN_FORCE_PHONE=` (vazio)
- `CAMPAIGN_MAX_CONTACTS` conforme rampa
